# Smart-Student-Management
Python console-based student management system to add, view, search, calculate results, and manage students with auto-roll numbers
# Smart Student Management System

This is a Python-based project for managing student records, calculating results, and tracking toppers. It is designed for beginners and intermediate Python learners to practice their skills.

## Features

- Add student records with multiple subjects
- View all student records
- Search for a student by roll number
- Calculate total marks, percentage, and grade
- Display the topper
- Delete a student or clear all records
- Automatic roll number reassignment after deletion

## How to Use

1.Clone this repository:
   ```bash
   git clone https://github.com/munazirabbas47-dotcom/Smart-Student-Management.git

2.Navigate to the project folder:
cd Smart-Student-Management

3.Run the main Python script:
python SmartStudentManagementSystem.py

4.Follow the on-screen menu to manage students. 

Special Notes:
    Roll numbers are automatically assigned and re-adjusted after deletion
    Only Python and standard libraries are required
    Ideal for practicing Python dictionaries, lists, loops, and functions
